## Setup:
#### Installation
Run
```
npm install
```
Your app is now installed and ready to go.
#### Build
Run
```
npm run build
```
Result of your build will be in ``build`` directory.
#### Live server
Run
```
npm start
```
Your live server is available at ``http://localhost:3000/``.
#### Tests
Run
```
npm run test
```
Tests report will be displayed in console.
